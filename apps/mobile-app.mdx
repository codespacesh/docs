title: Mobile app
description: iOS and Android app for Codespace
---

The Codespace mobile app lets you manage your resources and workspaces on the go.

## Features

- View resource status and details
- Browse workspaces across resources
- Start and stop workspaces
- Monitor provisioning progress

## Supported platforms

- **iOS** — iPhone and iPad
- **Android** — phones and tablets

## Authentication

The mobile app uses secure token storage:

- Sign in with your Codespace email and password
- Your session token is stored securely using the device's secure storage (Keychain on iOS, Keystore on Android)
- You stay signed in until you explicitly sign out

## Getting started

1. Install the app from the App Store or Google Play
2. Open the app and sign in with your Codespace credentials
3. Your resources and workspaces appear on the home screen

## Available actions

| Action | Description |
|--------|-------------|
| View resources | See all your resources with status |
| View workspaces | Browse workspaces in each resource |
| Start/stop | Control workspace lifecycle |
| Status monitoring | Real-time updates on resource and workspace state |

<Note>
  Some features like creating resources and managing organization members are only available on the web dashboard.
</Note>
