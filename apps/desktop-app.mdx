The CodeWire desktop app is a native application built with Rust that provides quick access to your workspaces.

## Features

- Browse resources and workspaces
- Open terminals directly from the app
- Workspace status monitoring
- GitHub authentication via device flow

## Authentication

The desktop app uses **GitHub device flow** for authentication — no browser redirect needed:

1. Open the desktop app
2. Click **Sign in with GitHub**
3. A device code is displayed
4. Open the provided GitHub URL in your browser
5. Enter the code and authorize
6. The app automatically detects authorization

<Info>
  No client secret is needed for the device flow. Authentication is handled securely between the app and GitHub.
</Info>

## Installation

The desktop app is available for macOS. Download the latest release and drag it to your Applications folder.

### Building from source

If you have the source code:

```bash
# Development build
bun run desktop

# Release build
bun run desktop:build
```

## Workspace access

From the desktop app, you can:

- View all your resources and their workspaces
- See real-time workspace status (running, stopped, etc.)
- Open a terminal session directly
- Navigate to workspace apps
